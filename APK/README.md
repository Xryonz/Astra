# Baixar o Astra

Downloads oficiais do Astra. Os arquivos ficam nos **[Releases do GitHub](https://github.com/Xryonz/Astra/releases/latest)** (não no repositório — assim o clone continua leve e é de lá que o app se **atualiza sozinho**).

---

## 🖥️ Desktop (Windows)

**[⬇️ Baixar para Windows 10/11 (64-bit)](https://github.com/Xryonz/Astra/releases/latest)**

1. Baixe o `Astra-<versão>-win-x64.zip` do último release.
2. **Extraia a pasta inteira** onde quiser (ex: `Documentos\Astra`). Não rode de dentro do `.zip`.
3. Abra o `Astra.exe`.
4. Na primeira vez o Windows pode mostrar o SmartScreen (app sem assinatura): **Mais informações → Executar assim mesmo**.

> **Atualização automática:** ao abrir, o Astra verifica se há versão nova e baixa sozinho — é só reiniciar quando ele avisar. Você não precisa voltar aqui.
